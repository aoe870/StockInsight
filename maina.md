### 6. 实时行情

获取股票实时行情数据。

- **接口地址**: `/stock/v2/realtime`
- **完整URL**: `https://miana.com.cn/api/stock/v2/realtime`

**请求参数**:

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| `token` | 字符串 | 是 | 通过登录获取的Token |
| `symbol` | 字符串 | 是 | 股票代码，支持多支股票用英文逗号隔开（一次最多20支） |
| `format` | 字符串 | 否 | 返回格式，`json`或`csv`（默认json） |

**请求示例**:
```
GET https://miana.com.cn/api/stock/v2/realtime?token=YOUR_TOKEN&symbol=sh600519,sz000001,bj920106,hk00700,usAAPL
```

**响应字段**:

| 字段名 | 类型 | 描述 |
|--------|------|------|
| `type` | 字符串 | 类型（STOCK：股票，INDEX：指数，FUND：基金，FOREX：外汇，CRYPTO：加密货币） |
| `name` | 字符串 | 股票名称 |
| `code` | 字符串 | 股票代码 |
| `countryCode` | 字符串 | 国家/地区代码 |
| `exchangeCode` | 字符串 | 交易所代码 |
| `chineseName` | 字符串 | 中文名称（美股） |
| `date` | 字符串 | 更新时间 |
| `price` | 字符串 | 当前价格 |
| `preClose` | 字符串 | 昨收价 |
| `open` | 字符串 | 开盘价 |
| `high` | 字符串 | 最高价 |
| `low` | 字符串 | 最低价 |
| `volume` | 字符串 | 成交量 |
| `amount` | 字符串 | 成交额 |
| `change` | 字符串 | 涨跌额 |
| `changeRate` | 字符串 | 涨跌幅 |
| `sellVolume` | 字符串 | 卖盘量 |
| `buyVolume` | 字符串 | 买盘量 |
| `circulationValue` | 字符串 | 流通市值 |
| `marketValue` | 字符串 | 总市值 |
| `circulationShares` | 字符串 | 流通股本 |
| `totalShares` | 字符串 | 总股本 |
| `committee` | 字符串 | 委比 |
| `pb` | 字符串 | 市净率 |
| `turnover` | 字符串 | 换手率 |
| `pe_ttm` | 字符串 | TTM市盈率 |
| `pe_dyn` | 字符串 | 动态市盈率 |
| `pe_static` | 字符串 | 静态市盈率 |
| `amplitude` | 字符串 | 振幅 |
| `highLimit` | 字符串 | 涨停价格 |
| `lowLimit` | 字符串 | 跌停价格 |
| `sells` | 数组 | 卖盘档位 |
| `buys` | 数组 | 买盘档位 |

### 8. K线行情

获取股票K线数据。
## 股票K线接口

| 项目 | 内容 |
|:---|:---|
| **接口名称** | 股票K线 |
| **请求方式** | GET |
| **接口地址** | `https://miana.com.cn/api/stock/v2/kline` |

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|:---|:---|:---|:---|
| token | string | * | token，通过登录获取 |
| symbol | string | * | 股票代码，上证用sh开头，深证用sz开头，北证用bj开头，港股用hk开头，美股用us开头 |
| type | string | * | k线类型：day（当日1分钟）、day5（最近5个交易日1分钟）、5min（5分钟K线）、10min（10分钟K线）、15min（15分钟K线）、20min（20分钟K线）、30min（30分钟K线）、60min（60分钟K线）、d1（日K）、w1（周K）、m1（月K）、y1（年K） |
| beginDate | 日期 |  | 起始日期，格式"yyyy-mm-dd hh:mm:ss" |
| endDate | 日期 |  | 结束日期，格式"yyyy-mm-dd hh:mm:ss" |
| order | string |  | 顺序：ASC表正序，DESC表倒序（默认在不输入时间为倒序，只输入开始时间为正序，只输入结束时间为倒序，输入开始时间和结束时间为倒序） |
| limit | 整数 |  | 数量，默认1000（最大为2000） |
| fq | string |  | 复权：默认不复权（fq表不复权，qfq表示前复权） |
| format | string |  | 格式，支持json格式和csv格式（默认json格式） |

### 使用说明
目前5min、10min、30min、60min、90min、120min数字只支持A股不复权数据。

### 适用版本
按量计费、免费版（不含分钟级数据）、个人版（不含分钟级数据）、专业版、企业版

### 完整示例URL

```
https://miana.com.cn/api/stock/v2/kline?token=5118a66a405a1278375862efbc69383b&symbol=sh600519&type=day
```

| high | 字符串 | 最高价 |
| low | 字符串 | 最低价 |
| price | 字符串 | 当前价 |
| volume | 字符串 | 成交量 |
