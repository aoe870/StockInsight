## 基础信息

### 数据覆盖范围
- **全球覆盖**：30+ 国家（地区）
- **交易所**：40+ 交易所
- **历史数据**：30+ 年历史数据
- **股票**：100,000+
- **指数**：500+
- **ETF**：15,000+
- **外汇**：800+
- **加密货币**：1,000+

### 接口通用规范
- **基础URL**: `https://miana.com.cn/api`
- **请求方式**: GET
- **认证方式**: Token（通过登录获取）
- **返回格式**: JSON / CSV（默认JSON）
- **字符集**: UTF-8

## 股票接口

### 1. 股票交易所
获取支持的交易所列表。

### 2. 股票清单
获取股票列表。

### 3. A+H股清单（港股）

### 4. 股票排行
获取股票排行数据。

### 5. IPO新股上市
获取新股上市信息。

### 6. 实时行情
获取股票实时行情数据。

**接口地址**：
```
https://miana.com.cn/api/stock/v2/realtime
```

**请求参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| token | 字符串 | 是 | 通过登录获取的Token |
| symbol | 字符串 | 是 | 股票代码，支持多支股票用英文逗号隔开（一次最多20支） |
| format | 字符串 | 否 | 返回格式，`json`或`csv`（默认json） |

**请求示例**：
```
GET https://miana.com.cn/api/stock/v2/realtime?token=YOUR_TOKEN&symbol=sh600519,sz000001,bj920106,hk00700,usAAPL
```

**响应字段**：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| type | 字符串 | 类型（STOCK：股票，INDEX：指数，FUND：基金，FOREX：外汇，CRYPTO：加密货币） |
| name | 字符串 | 股票名称 |
| code | 字符串 | 股票代码 |
| countryCode | 字符串 | 国家/地区代码 |
| exchangeCode | 字符串 | 交易所代码 |
| chineseName | 字符串 | 中文名称（美股） |
| date | 字符串 | 更新时间 |
| price | 字符串 | 当前价格 |
| preClose | 字符串 | 前收价 |
| open | 字符串 | 开盘价 |
| high | 字符串 | 最高价 |
| low | 字符串 | 最低价 |
| volume | 字符串 | 成交量 |
| amount | 字符串 | 成交额 |
| change | 字符串 | 涨跌额 |
| changeRate | 字符串 | 涨跌幅 |
| sellVolume | 字符串 | 卖盘量 |
| buyVolume | 字符串 | 买盘量 |
| circulationValue | 字符串 | 流通市值 |
| marketValue | 字符串 | 总市值 |
| circulationShares | 字符串 | 流通股本 |
| committee | 字符串 | 委比 |
| pb | 字符串 | 市净率 |
| turnover | 字符串 | 换手率 |
| pe_ttm | 字符串 | TTM市盈率 |
| pe_dyn | 字符串 | 动态市盈率 |
| pe_static | 字符串 | 静态市盈率 |
| amplitude | 字符串 | 振幅 |
| highLimit | 字符串 | 涨停价格 |
| lowLimit | 字符串 | 跌停价格 |
| sells | 数组 | 卖盘档位 |
| buys | 数组 | 买盘档位 |

## 板块接口

### 1. 板块清单
获取板块列表。

**接口地址**：
```
https://miana.com.cn/api/sector/v2/list?token=${token}
```

**请求参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| token | 字符串 | 是 | 通过登录获取的Token |
| type | 字符串 | 否 | 板块类型：`industry`(行业板块)、`concept`(概念板块)、`region`(地区板块)，默认为 industry |

**响应字段**：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| code | 字符串 | 板块代码 |
| name | 字符串 | 板块名称 |
| type | 字符串 | 板块类型：industry/concept/region |

**响应示例**：
```json
{
  "code": 200,
  "msg": "ok",
  "data": [
    {
      "code": "BK0737",
      "name": "CAR-T细胞疗法",
      "type": "CONCEPT",
      "countryCode": "CHN"
    },
    {
      "code": "BK1166",
      "name": "软件开发",
      "type": "CONCEPT",
      "countryCode": "CHN"
    }
  ]
}
```

### 6. 板块K线行情
获取板块K线数据。

**接口地址**：
```
https://miana.com.cn/api/sector/v2/kline?token=${token}&sector=${sector_code}&type=${type}&beginDate=${date}&endDate=${date}
```

**请求参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| token | 字符串 | 是 | 通过登录获取的Token |
| sector_code | 字符串 | 是 | 板块代码 |
| type | 字符串 | 否 | K线类型：`day`(当日1分钟）、`day5`(最近5个交易日1分钟）、`5min`(5分钟K线）、`10min`(10分钟K线）、`15min`(15分钟K线）、`30min`(30分钟K线）、`60min`(60分钟K线）、`d1`(日K）、`w1`(周K)、`m1`(月K)、`y1`(年K) |
| beginDate | 日期 | 否 | 起始日期，格式：`yyyy-mm-dd hh:mm:ss` |
| endDate | 日期 | 否 | 结束日期，格式：`yyyy-mm-dd hh:mm:ss` |

**响应字段**：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| date | 日期 | 更新时间 |
| open | 数值 | 开盘价 |
| high | 数值 | 最高价 |
| low | 数值 | 最低价 |
| price | 数值 | 收盘价 |
| volume | 数值 | 成交量 |
| amount | 数值 | 成交额 |

**适用版本**：
- 5min、10min、15min、30min、60min：专业版、企业版
- d1、w1、m1：免费版、个人版
